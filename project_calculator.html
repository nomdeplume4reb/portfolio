REACTulator!

Calculator built with REACT


This project uses REACT.js, HTML, and CSS to design a basic calculator that contains the following:

  • 10 clickable elements containing one number each from 0-9
  • 4 clickable elements each containing one of the 4 primary mathematical operators
  • 1 clickable element containing a decimal point symbol
  • 1 clickable element with an id="clear"
  • an element to display values
  • a clear button to clear the input and output values and return the calculator to its initialized state

    • The calculator should not allow a number to begin with multiple zeros, and it should have several decimal places of precision when it comes to rounding.
